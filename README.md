# Sticky Social Media Navigation Bar

Um componente de barra de navegação lateral (Sticky/Floating Sidebar) moderno, interativo e totalmente responsivo desenvolvido com HTML5, CSS3 e JavaScript. O projeto apresenta uma barra flutuante fixada na lateral da tela contendo ícones das principais redes sociais, controlada por um botão dinâmico de ativação (toggle) que expande ou recolhe o menu com animações fluidas.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica da lista de links sociais (`<a>`), contêiner flutuante e botões de controle.
* CSS3: Estilização completa, uso de cantos suavizados, posicionamento fixo (`position: fixed`) e gerenciamento da identidade cromática de cada marca.
* JavaScript: Lógica de controle de estados (abrir/fechar), escuta de eventos de clique e alternância dinâmica de classes no DOM para disparar as transições.

## Funcionalidades e Técnicas Aplicadas

* Menu Lateral Retrátil (Floating Toggle): Inclusão de um botão flutuante vermelho com ícone de fechar (`✕`) ou abrir na base, que serve como gatilho para exibir ou ocultar romanticamente a barra de redes sociais.
* Paleta de Cores Institucionais Integrada: Cada botão quadrado possui cantos arredondados e é personalizado estritamente com a cor oficial de sua respectiva plataforma (Pinterest, LinkedIn, YouTube, Dribbble, GitHub, Facebook e Twitter).
* Posicionamento Fixo Estratégico: Uso da propriedade `position: fixed` no canto esquerdo da viewport, garantindo que o componente acompanhe a rolagem da página (*sticky effect*) e permaneça sempre acessível ao utilizador.
* Microinterações e Transições Suaves: Animações em CSS puro integradas a propriedades de opacidade, deslocamento (`transform: translateX`) e transições de tempo para criar um efeito de surgimento sofisticado.
